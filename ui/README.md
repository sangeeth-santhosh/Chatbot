# Chat UI

Local React, Vite, Tailwind CSS, Zustand, Axios, React Router, and Socket.IO client for the realtime chat platform.

## Setup

```bash
npm install
npm run dev
```

The UI runs only on:

```text
http://localhost:5173
```

The UI calls the backend at:

```text
http://localhost:5000
```

## Scripts

- `npm run dev`
- `npm run build`
- `npm run lint`
- `npm run preview`
